# 🎮 VALORANT OBS Rank Overlay Generator

VALORANTのゲーム内ランクとRR（ランクレーティング）をリアルタイムで取得し、OBSなどの配信ソフトにスタイリッシュに表示するためのオーバーレイ生成ツールです。

他の配信者や視聴者の方も、ご自身のRiot IDとタグを入力するだけで、自分専用のカスタムトラッカーURLを簡単に作成できます。

---

## ✨ 主な機能
- **簡単URL生成**: 画面上のフォームにRiot ID、タグを入力し、テーマカラーを選ぶだけでOBS用のURLが完成します。
- **配信映えデザイン**: VALORANTのサイバーな世界観にマッチする、ネオン光彩付きのスタイリッシュなバッジデザイン。
- **公式ランクアイコン対応**: ランクの文字だけでなく、現在のランクに応じた公式のアイコン画像が自動で表示されます。
- **自動更新（2分ごと）**: 配信中に手動でリロードしなくても、試合が終われば自動的に最新のランク・RRに切り替わります。
- **背景透過対応**: OBSの「ブラウザソース」に貼り付けるだけで、背景は自動的に透明になります。

---

## ⚙️ OBSへの設定方法・使い方

1. 本ツールのWebサイトにアクセスします。
2. 設定フォームに、あなたの **「Riot ID（ゲーム内の名前）」** と **「タグ（#より後ろの英数字）」** を入力します。
3. お好みに合わせて「テーマカラー（枠線の色）」をカラーパレットから選択します。
4. リアルタイムプレビューで見た目を確認し、問題なければ **「URLをコピー」** ボタンを押します。
5. OBS Studioを開き、ソース一覧の **「＋」ボタン ＞ 「ブラウザ」** を追加します。
6. コピーしたURLをURL欄に貼り付けます。
7. OBS側のサイズ設定を **幅: 350、高さ: 100** に変更すると、綺麗にバッジが収まります。

---

## ⚠️ 注意事項・うまく表示されない場合
- **戦績の公開設定を確認してください** VALORANTのインゲーム設定、または「Tracker.gg」などの外部サイトで、ご自身の戦績（プロファイル）が **「公開（Public）」** になっている必要があります。非公開設定になっていると、データを取得できずエラー（CHECK SETTINGS）が表示されます。
- **Riot ID・タグの入力ミス** 全角・半角の打ち間違いや、タグに「#」を入れてしまっていないか確認してください。
